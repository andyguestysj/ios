IOS Game Development

This module will provide students with the space to learn, explore and absorb the core concepts, principles and techniques of creating games for apple mobile devices. The module will guide students through the core principles of programming in apple specific languages. Starting with an introduction to the apple programming environment, the module will progress delving into the various specifics of apples games elated frameworks and languages. The aim is to provide a comprehensive introduction to games programming for iOS enabled devices.

Game Engines

From libraries to extend programming libraries to full fledged game engines there are many tools to help us make games.

Engine Components

Game Engines consist of components that implement rendering, sound, AI, physics, game logic and other features.

Better Games

Even if you don't go on to develop game engines, a better understanding of how they are developed will help you make better games.
